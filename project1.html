Game Scorer

C Language

In 2023, I developed a straightforward program in the C language. This program allows users to input player names and generates a random tournament schedule. Users can then enter scores for each match, and at the end of each game and tournament, the scores and winners are displayed. Additionally, the program utilizes file handling to save all data to a text file for future reference. This project was undertaken as part of my personal practice and skill development in programming.
